# FAST API com Python

FAST API criada com Python para operações de CRUD, usando MySQL como banco de dados.

## 🛠️ Tecnologias
- FAST API
- Uvicorn
- Pydantic
- Sqlalchemy

## 🚀 Como rodar

1.Ter o Python instalado
2.Fazer o Download do Uvicorn,Pydanic e Sqlalchemy
3.Ter o MySQL instalado 
4.Instalar o Postman para testes
5.Configurar o banco de dados no arquivo database.py

## 📋 Endpoints

| Método | Rota | Descrição |
|--------|------|-----------|
| GET | /tarefas | Recupera lista de tarefas |
| POST | /tarefas | Cria uma nova tarefa |
| PUT | /tarefas | Atualiza uma tarefa pelo id |
| DELETE | /tarefas | Exclui uma tarefa |
